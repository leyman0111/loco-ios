# Loco iOS - Инструкция по настройке

## Этап 1: Экран авторизации ✅

Реализован экран авторизации с интеграцией Яндекс OAuth.

### Что реализовано:

1. **LoginView.swift** - экран авторизации с дизайном:
   - Пастельный фон (кремовый #FCFBF4)
   - Декоративные круги (голубой и золотой)
   - Логотип Loco с иконкой геолокации
   - Поля ввода Email/Password
   - Кнопка Login (коралловый цвет)
   - Ссылка "Sign up"
   - OAuth кнопки (Google, Яндекс, VK)

2. **YandexOAuthService.swift** - сервис для авторизации через Яндекс:
   - Использует `ASWebAuthenticationSession`
   - Открывает браузер для авторизации
   - Получает access token из callback URL

3. **AuthViewModel.swift** - логика авторизации:
   - Обработка успешной авторизации через Яндекс
   - Показ alert с токеном: "Авторизовано в Яндекс! {token}"
   - Заглушки для Google и VK: "Этот функционал в стадии разработки. Подождите немного, пожалуйста))"

4. **LocoTheme.swift** - система дизайна:
   - Цвета из макета
   - Типография
   - Отступы и радиусы

### Настройка Яндекс OAuth:

1. **Зарегистрируйте приложение** на https://oauth.yandex.ru/client/new

2. **Настройте Redirect URI**:
   - Добавьте: `loco://oauth/yandex`

3. **Получите Client ID** и замените в `YandexOAuthService.swift`:
   ```swift
   private let clientId = "YOUR_YANDEX_CLIENT_ID" // ← Вставьте ваш Client ID
   ```

4. **URL Scheme уже настроен** в `Info.plist`:
   - Scheme: `loco`
   - Callback URL: `loco://oauth/yandex`

### Как запустить:

1. Откройте `loco-ios.xcodeproj` в Xcode
2. Выберите симулятор (iPhone 15 Pro или любой другой)
3. Нажмите **Cmd + R** для запуска
4. Приложение откроется с экраном авторизации

### Тестирование:

- **Яндекс OAuth**: Нажмите на желтую кнопку "Я" → откроется браузер → авторизуйтесь → вернётесь в приложение → увидите alert с токеном
- **Google**: Покажет "Этот функционал в стадии разработки..."
- **VK**: Покажет "Этот функционал в стадии разработки..."
